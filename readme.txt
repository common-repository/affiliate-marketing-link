== Description ==

安装激活插件后，当用户点击您博客内的新蛋、卓越亚马逊、京东、一号店、凡客诚品等支持CPS的链接时，此插件将在链接跳转到商家时自动加上您在后台设置的对应网站联盟ID，进而为您网站带来收益！
更详细描述请移步阅读 http://labs.gokesoft.com/affiliate-marketing-link/
when user click affiliate marketing Link on your site, This plugin auto add your union ID.

== Installation ==

在wordpress管理后台 - 安装新插件 - 搜索 “Affiliate Marketing Link”，点击安装
或者直接本插件，下载完成解压缩后，将整个Affiliate-Marketing-Link上传到您博客\wp-content\plugins\目录下
